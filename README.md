### 🔭 Featured Side Projects

Here is what I'm currently building and iterating on:

* 🤖 **[autoresearch-win-rtx](https://github.com/jsegov/autoresearch-win-rtx)** * **What it is:** A recognized community variant of Andrej Karpathy's `autoresearch` repo, adapted for native Windows support on consumer NVIDIA GPUs.
  * **Highlights:** 260+ stars. Introduces CUDA-based optimizations and hardware-aware runtime tuning (tiered VRAM floors for Turing/Ampere/Ada/Blackwell architectures) for local RTX-class training.

* 🔌 **[shipspec-claude-code-plugin](https://github.com/jsegov/shipspec-claude-code-plugin)** * **What it is:** A Spec-Driven Development Plugin for Claude Code.
  * **Highlights:** Solves AI "vibe coding" drift by converting feature ideas into structured PRDs, SDDs, and executable implementation plans. Uses a "Ralph Wiggum loop" for built-in task verification.

* 🚢 **[shipspec-cli](https://github.com/jsegov/shipspec-cli)** * **What it is:** An AI-native CLI for codebase analysis and production readiness planning. 
  * **Highlights:** Uses AST-based semantic chunking (Tree-sitter), local vector embeddings (LanceDB), and LangGraph orchestration to generate readiness reports, PRDs, and task graphs. Features an interactive terminal UI (OpenTUI).

* 🌐 **[segov-dev](https://github.com/jsegov/segov-dev)** * **What it is:** My terminal-inspired personal portfolio and blog.
  * **Highlights:** Built with Next.js 15, React 19, and Tailwind CSS. Features a custom AMA chatbot powered by Vercel AI SDK Agents, Vercel Blob (for resume RAG retrieval), and Edge Config.
